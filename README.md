# 🚚 Smart Logistics Optimization Dashboard

A real-time logistics optimization system demonstrating core algorithmic paradigms on a delivery network.

---

## 🎯 Project Overview

This project simulates a smart logistics system where delivery requests are processed, optimized, and visualized through a dynamic dashboard.

It integrates multiple algorithmic techniques into a single pipeline:
- Sorting for prioritization
- Graph algorithms for routing
- Greedy methods for fast decisions
- Dynamic Programming for optimal solutions
- Divide & Conquer for scalability

---

## ⚙️ Key Features

| Feature | Description |
|--------|------------|
| 📦 Delivery Prioritization | Sort packages based on deadlines |
| 📊 Real-Time Dashboard | Visualizes outputs using charts |
| ⚡ Greedy vs DP Comparison | Shows difference between fast and optimal solutions |
| 🗺️ Route Optimization | Shortest path using Dijkstra |
| ⏱️ Live Simulation | Deliveries processed step-by-step |
| 📈 Data Visualization | Chart.js graphs for analysis |

---

## 🧠 Algorithm Modules

### 🔹 Module 1 — Sorting
- Merge Sort → stable sorting by deadline
- Quick Sort → priority-based ranking
- Heap Sort → dynamic priority queue

---

### 🔹 Module 2 — Graph Algorithms
- Dijkstra → shortest path optimization
- BFS → connectivity validation

---

### 🔹 Module 3 — Greedy
- Fractional Knapsack → fast value maximization
- Activity Selection → maximize deliveries

---

### 🔹 Module 4 — Dynamic Programming
- 0/1 Knapsack → optimal delivery selection
- Ensures global optimum over greedy approximation

---

### 🔹 Module 5 — Divide & Conquer
- Merge Sort scalability
- Handles large datasets efficiently

---

## 📊 Real-Time Simulation

The system includes a simulation layer where:
- Deliveries appear sequentially
- Charts update dynamically
- Algorithm outputs are visualized live

This mimics real-world logistics processing systems.
